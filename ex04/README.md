# 第四回
## 逃げろこうかとん(ex04/dodge_bomb.py)
### ゲーム概要
- ex04/dodge_bomb.pyを実行すると、1600×900のスクリーンに草原が描画され、こうかとんを移動させ土びまわる爆弾から逃げるゲーム
- こうかとんが爆弾と接触するとゲームオーバーで終了する
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
- こうかとんの初期位置がランダムで設定
- 四角い爆弾を追加
